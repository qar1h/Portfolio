My Projects

Project 1:Guitar Database Website

One of the project that I want to work on is a Guitar Database Website. It will be an interactive platform designed for guitarists of all levels, providing a comprehensive resource for chords, scales, and chord progressions for different tunings. This project aims to enhance the musical experience for guitar players by offering a well-organized, searchable database of essential information for making compostions which otherwise would be difficult to theory-craft on their own.

Project 2:AI Image Generator

Another project that I want to make in the future is an AI Image Generator as it is an innovative web application that utilizes advanced artificial intelligence algorithms to create unique and stunning images based on user input. This project aims to provide users with an intuitive platform to generate images that suit their creative needs, whether for personal use, social media, or professional projects.
